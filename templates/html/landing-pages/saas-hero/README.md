# Landing Page SaaS Hero

## Descrição
Landing page responsiva com seção hero, prova social e CTA duplo.

## Tecnologias Utilizadas
- HTML5
- Tailwind CSS
- Alpine.js

## Pré-requisitos
- Node.js 18+

## Instalação
```bash
npm install
npm run dev
```

## Customização
Cores em `src/styles/tailwind.config.js`.
