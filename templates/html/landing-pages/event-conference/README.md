# Experience Conference 2024

## Descrição
Landing page para eventos e conferências de tecnologia, com foco em conversão e destaque para palestrantes, agenda e venda de ingressos.

## Tecnologias Utilizadas
- HTML5 semântico
- CSS3 moderno (Grid/Flexbox)
- Google Fonts
- Font Awesome (ícones sociais)

## Pré-requisitos
- Navegador moderno (Chrome, Firefox, Edge, Safari)
- Ferramenta de build opcional (Parcel/Vite) caso deseje integrar bundlers

## Instalação
```bash
cp -R templates/html/landing-pages/event-conference novo-projeto
cd novo-projeto
```

## Configuração
- Atualize o arquivo `index.html` com as informações de data, local e CTA.
- Substitua as imagens em `assets/`.

## Estrutura de Pastas
- `index.html` – markup principal
- `styles.css` – estilos base mobile-first
- `assets/` – imagens dos palestrantes e background

## Customização
- Paleta de cores definida em `:root` no `styles.css`
- Tipografia controlada pelo Google Fonts (Poppins)
- Componentes organizados por seções (hero, about, speakers, agenda, tickets)

## Demonstração
- Adicione screenshots dentro da pasta `assets/preview/`
