# Starters Next.js

## Descrição
Coleção de starters Next.js com rotas app router, SSR e suporte a SEO.

## Tecnologias Utilizadas
- Next.js 13+
- TypeScript
- Tailwind CSS
- Prisma

## Pré-requisitos
- Node.js 18+
- PostgreSQL ou MySQL

## Instalação
```bash
npm install
npm run dev
```

## Configuração
- Defina variáveis no `.env`
- Execute `npx prisma migrate dev`

## Estrutura
- `starters/saas`
- `starters/landing`
- `starters/dashboard`

## Customização
Edite `theme.config.ts` e tokens de Tailwind em `tailwind.config.js`.
