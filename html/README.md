# Templates HTML

## Descrição
Landing pages, dashboards e sites estáticos otimizados para conversão e performance.

## Tecnologias Utilizadas
- HTML5, CSS3
- Sass/SCSS
- Vanilla JS
- Bootstrap 5 / Tailwind

## Pré-requisitos
- Node.js 18+
- npm ou yarn

## Instalação
```bash
npm install
npm run dev
```

## Configuração
- Variáveis de ambiente em `.env`
- Ajuste cores no arquivo `styles/_variables.scss`

## Estrutura de Pastas
- `landing-pages/`
- `dashboards/`
- `portfolios/`
- `corporativo/`
- `components/`

## Customização
Cada template possui notas no topo do HTML sinalizando seções editáveis.
